本文最后更新于 2025-12-06 10:55:05

22 年 7 月底开始接触 Obsidian，用它维护我的笔记本有 3 年多的时间了。一直构想搭建个人博客网站，在笔记本里写点东西发到网上。我的需求是：本地继续使用 Obsidian 维护笔记仓库，写完笔记以后能够直接通过几行命令发布博客，最好不要配置图床。最近摸索出了一套对我来说几乎完美的零成本解决方案。

Obsidian 撰写维护本地笔记，使用 Hexo 构建网页，并上传到 Github 发布

  1. Obsidian 本地撰写笔记
  2. Hexo 将笔记转换成网页文件，并上传到 Github
  3. Github 将网页文件渲染成网页

obsidian 中的笔记以 .md 格式保存，Hexo 将 .md 文件转换成 html 格式，并且通过 git 上传到 Github 的特殊仓库，这个仓库名称必须是 <username>.github.io。

Hexo + Obsidian 完美维护博客的优点

  1. 图片保存在本地，上传到 Github，不用配置图床
  2. 博客上传方便，同步速度快
  3. 零成本
